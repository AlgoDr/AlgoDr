### 🧠 [Multimodal Identity RAG](https://github.com/AlgoDr/Multimodal-RAG)
Hybrid RAG over personal documents — PDFs and images in a unified index. Text queries route to BGE + Gemini LLM. Visual queries route to CLIP. Image content is searchable via Gemini Vision captions. Diagnosed and bypassed two upstream LlamaIndex retriever bugs. Agentic tool-calling with context memory built on LlamaIndex FunctionAgent.

`LlamaIndex` `BGE` `CLIP` `Gemini Vision` `PyMuPDF` `Tesseract` `Python 3.13`
